# Iteracja

Iteracja (łac. iteratio) - czyność powtarzania tej samej operacji w pętli z góry określoną liczbą razy lub aż do spełnienia określonego warunku.

Przykład iteracji.

```python
i = 0
while i < 10:
    i += 1
```
